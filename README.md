#Chrome-pwd

Lists chrome browser passwords

##Installation

#####OSX
```sh
make
```

#####Windows
```sh
mingw32-make -f Makefile.win
```

##Tests

#####OSX
```sh
make tests
```

##Usage

#####OSX
```sh
./build/cdr
```

#####Windows
```sh
build\cdr.exe
```
